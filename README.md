# Portafolio - Renato Alvarez Ortega

Portafolio personal y currículum vitae web con diseño moderno y responsivo.

## Características

- Modo claro/oscuro con persistencia
- Animaciones y transiciones suaves
- Galería de proyectos con lightbox
- Diseño responsivo (mobile-first)
- Despliegue automático con GitHub Actions

## Tecnologías

- HTML5
- CSS3 (variables, grid, flexbox)
- JavaScript vanilla
- GitHub Actions (CI/CD)

## Estructura

```
├── index.html          # Página principal
├── styles.css          # Estilos y temas
├── main.js             # Interactividad
├── assets/             # Imágenes de proyectos
└── .github/workflows/  # Despliegue automático
```

## Despliegue

El proyecto se despliega automáticamente a Easypanel cuando se hace push a `main`.

### Configuración

1. Agregar el secret `EASYPANEL_DEPLOY_URL` en GitHub (Settings > Secrets > Actions)
2. Push a la rama `main`
